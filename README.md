# Class exercise:  for-loop

```
for row in range(1,10)
    for colum in range(1,10)
        if colum>row
            print ('{}*{}={}'.format(row,colum,row*colum),end='\t')
        else: 
             print ('',end='\t')
    print('')  
   
```
